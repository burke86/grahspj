Nebular resources
=================

These compact NumPy resources are generated from the CIGALE v2022.1
`database_builder/nebular` source tables:

- `lines.dat`
- `line_wavelengths.dat`
- `continuum.dat`

The line wavelengths are stored in Angstrom and line luminosities are stored
on the full CIGALE `(Z, logU, ne, line)` grid per ionizing photon. The continuum
grid is stored as W/Angstrom per ionizing photon on the original CIGALE
continuum wavelength grid.
