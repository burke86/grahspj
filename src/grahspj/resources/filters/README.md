# Filter provenance

This directory contains vendored filter curve files redistributed with `grahspj`.

- Upstream project: `GRAHSP`
- Upstream license: `CeCILL v2`
- Local license copy: [CeCILL-v2.txt](/Users/colinburke/research/grahspj/LICENSES/CeCILL-v2.txt)
- Additional source: SVO Filter Profile Service for `CTIO/DECam.u`,
  `CTIO/DECam.g`, `CTIO/DECam.r`, `CTIO/DECam.i`, `CTIO/DECam.z`, and
  `CTIO/DECam.Y`.

The available filter names and relative data paths are listed in
`filter_registry.txt`. Filter curves are organized by facility/instrument
subdirectories.
